# NuclearGen

This repository contains four main components that can be run independently:

## Quick Start Guide

### Agent_1 (Node.js Application)
```bash
cd /Agent_1
npm install
node server.js
```

### Viz-Agent (Python Visualization)
```bash
cd /viz-agent
python agent.py
```

### MultiAgent System
```bash
cd /MultiAgent
./run.sh
# Or alternatively
python main.py
```

### Data Validation Tool
```bash
cd /data-validation
./run.sh
# Or alternatively
python app.py
```
